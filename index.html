Sync to Google Sheets

Paste your Google Apps Script web-app URL below, then sync either dataset to a connected Google Sheet.

Setup steps:
1. Open Google Sheets and create a new blank spreadsheet.
2. Click Extensions → Apps Script (this opens the editor linked to that sheet).
3. Replace all the default code with the script below.
4. Click Deploy → New deployment → Web app:
   • Execute as: Me
   • Who has access: Anyone
5. Grant permissions when prompted.
6. Copy the web-app URL and paste it below.
Note: If you later get a CORS error, you need to delete the old deployment and create a new one. Go to Deploy (top right) → Manage deployments → Click the three dots on the web app → Delete. Then create a new deployment following steps 4-6.
